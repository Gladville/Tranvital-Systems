# **App Name**: Tranvital Systems

## Core Features:

- Homepage Hero Section: Displays a split layout with a deep teal content block on the left and a professional image on the right. Overlapping elements create a modern look.
- Product Catalog: Presents laboratory equipment, medical equipment, and solar power solutions with clear headings, descriptions, and images. It features categorized display of all the available product with filtering.
- Services Showcase: Highlights installation and after-sales support, emphasizing technical competence and reliability.
- Contact Form with Location Map: Provides a minimalist contact form integrated with a map showing the company's location. Map view should provide satellite and street views, with zooming, and other navigation tools. It will be used to generate email upon submission.
- AI-Powered Product Finder: Tool to help customers find the perfect equipment based on their requirements, powered by a large language model using reasoning.

## Style Guidelines:

- Primary color: Deep Teal (#006D6F) for trust and stability.
- Accent color: Solar Gold (#006D6F) for CTAs and highlights, indicating energy and innovation.
- Neutral color: Black for body text and UI elements.
- Base color: Clean White (#FFFFFF) for clarity and hygiene.
- Brand Tint: Light Teal (#C8E0E8) for section backgrounds and cards.
- Headline font: 'Inter' (sans-serif) for large, confident headlines.
- Body text font: 'Source Sans Pro' (sans-serif) for generous line spacing and readable letterforms.
- Use overlapping boxes, layered panels, and asymmetry in hero sections.
- Use clear icons related to Lab, Medical, and Solar categories.
- Subtle animations (fade, slide, hover) for smooth transitions.
